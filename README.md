# TestAspWebApi

A simple ASP.NET MVC app with a web page and API for a desktop application, along with an example application itself. It supports user registration and authorization, using AspNetCore.Identity and a SQL database. There are two user authorization options used for the API: cookies and JWT tokens. It uses an authorization scheme with access tokens and refresh tokens.

.NET version: 8.0. Frontend: cshtml, bootstrap. Database: MSSQL/SQLite (can be selected in config file)
